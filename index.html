Mulai kembangkan server store mu dengan Pixel Bot!

Pixel Bot adalah bot discord yang dibuat sebagai tools untuk server store yang awalnya dibuat untuk Pixel Mart, namun kini Pixel Bot hadir ke publik untuk membantu server storemu dengan tools yang disediakan.

Add To Your Store

Feature

Auto Role

Auto Role merupakan fitur yang umum, dimana saat ada member yang baru bergabung maka member itu akan mendapatkan role yang sudah di tentukan. Gunakan command /auto-role untuk mengatur role yang akan ditetapkan.

Welcome Message

Welcome Message bot ini berbeda dengan bot yang lain, yang dimana Pixel Bot sendiri menggunakan embed untuk welcome message agar lebih simpel. Sayangnya yang bisa diubah dari welcome message hanyalah footernya. Kamu bisa mengatur footer menggunakan command /welcome-footer dan pilih templatenya, Gunakan command /welcome-channel untuk mengatur di channel mana welcome message akan dikirim.

Testimoni & Rating

Testimoni & Rating lah yang membuat bot ini menjadi prioritas server store karena command ini berfungsi untuk posting testimoni dan juga rating pembeli ke penjual yang melayaninya, fitur ini juga akan memberikan role buyer secara otomatis jika kamu sudah mengatur role buyer. Gunakan command /testimoni-channel dan /rating-channel untuk mengatur channelnya agar tidak bisa digunakan di channel lain.

Verify

Eitss, kalian harus cobain nih pake sistem verifikasi Pixel Bot, "loh kenapa?" karena Pixel Bot memiliki fitur verifikasi yang berbeda, yaitu menggunakan tombol dan akan disuruh memilih angka yang benar seperti verify captcha. Langsung aja cobain!. Gunakan command /verify-role untuk mengatur rolenya kemudian gunakan command !verifymsg di channel verifikasimu.

Ready to try Pixel Bot?

Beberapa fitur diatas merupakan sebagian fitur Pixel Bot, cobalah tanbahkan Pixel Bot ke server storemu kemudian gunakan command !setup-info untuk melihat apa saja fitur yang bisa digunakan. Yuk invite Pixel Bot sekarang!
